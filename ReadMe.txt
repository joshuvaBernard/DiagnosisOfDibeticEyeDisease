1.Create a Virtual Environment on where the files are located by using command prompt

command : python -m venv myenv

2.To activate Virtual environment

command : myenv\Scripts\activate

3.after activation install the neccessary packages

command : pip install -r requirements.txt


4.once finished installing the requirements
run main.py

command : python main.py
